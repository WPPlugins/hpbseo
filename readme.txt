== Description ==
It is plug-in which can perform a SEO measure on Wordpress. 
hpbdashboard is needed in order to operate this plug-in. 
If this plug-in is used, advice can be seen about a title, a meta-description, a meta-keyword, the text, keyword balance, etc. 

== Installation ==

1. Upload `hpbseo` folder and all its contents to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
